# Vault Protocol: Lockdown Escape

## Description
A text-based escape game written in C where the player navigates a locked facility, solves security puzzles, and attempts to escape.

## Features
- Multiple zones (Control Room, Security Office, Server Room)
- Authentication puzzles
- Player navigation system
- Win condition (escape)

## How to Run
1. Compile:
   gcc main.c -o game
2. Run:
   ./game

## Controls
- Enter numbers to make selections
- Follow on-screen instructions
